# Componentes

## Crear un componente
**⌥ ⌘ K** (Ctrl+Alt+K) — convierte marco/grupo en COMPONENT. Etiqueta morada con diamante.

## Conjuntos de componentes
**⇧ ⌘ K** — combina 2+ componentes con borde morado punteado.

## Crear instancias
Clic derecho → **Crear instancia**. Aparece 40 px a la derecha.

## Desenlazar una instancia
**⌥ ⌘ B** — se convierte en marco sin enlace.

## Sincronización en vivo
Editar un componente actualiza todas sus instancias. Propiedades sincronizadas: dimensiones, rellenos, trazos, efectos, opacidad, radios de esquinas, layout.

## Overrides
Las instancias pueden sobreescribir propiedades sin romper el enlace.

## Hit testing
Clic selecciona el componente. **Doble clic** para entrar y seleccionar hijos.

| Acción | Mac | Windows / Linux |
|--------|-----|-----------------|
| Crear componente | ⌥ ⌘ K | Ctrl + Alt + K |
| Crear conjunto | ⇧ ⌘ K | Shift + Ctrl + K |
| Desenlazar instancia | ⌥ ⌘ B | Ctrl + Alt + B |
