## Description
Créer apps/site/ avec Nuxt ou Next.js et développer la landing page complète de verso.dev : hero, features, how it works, pricing, footer.

## Tâches
- [ ] Initialiser apps/site/ avec le framework choisi (Nuxt ou Next.js)
- [ ] Configurer le routing, le SEO, et les métadonnées Open Graph
- [ ] Section Hero : titre accrocheur, sous-titre, CTA, animation/illustration
- [ ] Section Features : 6-8 features clés avec icônes et descriptions
- [ ] Section "How it works" : 3 étapes visuelles du workflow IA
- [ ] Section Pricing : Free (solo), Pro ($19/mo), Team ($49/seat/mo)
- [ ] Section Testimonials/Social proof (placeholder pour le lancement)
- [ ] Footer : liens, newsletter signup, réseaux sociaux, mentions légales
- [ ] Design responsive mobile-first
- [ ] Optimisation performance (Lighthouse > 95)

## Validation
- Landing page complète et visuellement aboutie
- Responsive sur mobile, tablette et desktop
- Lighthouse score > 95 (performance, accessibility, SEO)
- CTA fonctionnels (liens vers download, docs, pricing)
- SEO : meta tags, Open Graph, sitemap
