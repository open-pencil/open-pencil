# useScrubInput

`useScrubInput()` lee el contexto local del scrub input proporcionado por `ScrubInputRoot`.

Úsalo dentro de `ScrubInputField`, `ScrubInputDisplay` o descendientes personalizados que necesiten el estado de edición y arrastre.
