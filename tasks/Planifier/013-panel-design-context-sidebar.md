## Description
Panel complet dans la sidebar Vue 3 pour gérer le Design Context : brand (couleurs, typo, ton), style (direction artistique), audience (personas, devices), references (URLs, captures), rules (contraintes métier).

## Tâches
- [ ] Refondre le DesignContextPanel avec un design Verso finalisé
- [ ] Brand : color picker avancé, font selector, tone slider
- [ ] Style : mood board avec drag & drop d'images, tags de style
- [ ] Audience : formulaire personas avec avatars, device picker
- [ ] References : URL previews avec Open Graph, galerie de captures
- [ ] Rules : éditeur de règles avec syntaxe simplifiée, toggle on/off
- [ ] Sync temps réel avec le store et le fichier .design
- [ ] Animations de transition entre sections (accordéon fluide)
- [ ] Mode compact pour les écrans étroits
- [ ] Raccourci clavier pour ouvrir/fermer le panel

## Validation
- Panel complet avec les 5 sections
- Design conforme à la charte Verso
- Sync bidirectionnelle avec le store
- Raccourci clavier fonctionnel
- Responsive sur écrans de 1280px à 2560px
