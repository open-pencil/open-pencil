# useLayerTree

`useLayerTree()` legge il contesto dell'albero dei layer locale fornito da `LayerTreeRoot`.

Usalo all'interno dei discendenti personalizzati dell'albero dei layer che hanno bisogno di accedere agli elementi dell'albero, allo stato della selezione, allo stato di espansione o alle azioni di riga.
