# useScrubInput

`useScrubInput()` legge il contesto dello scrub input locale fornito da `ScrubInputRoot`.

Usalo all'interno di `ScrubInputField`, `ScrubInputDisplay` o discendenti personalizzati che hanno bisogno dello stato di modifica e scrubbing.
