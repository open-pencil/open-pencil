## Description
Polish complet de l'UI Verso : refonte visuelle des panels, splash screen animé, onboarding first-run, raccourci Cmd+Shift+V, menu Verso custom, page settings.

## Tâches
- [ ] Refonte visuelle de tous les panels (cohérence typographie, spacing, couleurs)
- [ ] Splash screen animé avec logo Verso au démarrage desktop
- [ ] Onboarding first-run : 3-5 étapes pour découvrir l'éditeur
- [ ] Raccourci Cmd+Shift+V (Ctrl+Shift+V) pour ouvrir la palette de commandes Verso
- [ ] Menu "Verso" custom dans la barre de menu (About, Check for updates, Settings)
- [ ] Page Settings : thème éditeur, langue, raccourcis, MCP config, export defaults
- [ ] Micro-interactions : hover states, focus rings, transitions, animations subtiles
- [ ] Tooltips contextuels sur tous les boutons/icônes
- [ ] État empty pour chaque panel (message + illustration quand vide)
- [ ] Vérifier la cohérence visuelle globale de A à Z

## Validation
- Tous les panels visuellement cohérents avec la charte Verso
- Onboarding complet et skip possible
- Settings fonctionnels et persistés
- Raccourci Cmd+Shift+V ouvre la palette de commandes
- Aucun état vide sans message explicatif
