# useScrubInput

`useScrubInput()` liest den lokalen Scrub-Eingabe-Kontext, der von `ScrubInputRoot` bereitgestellt wird.

Verwenden Sie es innerhalb von `ScrubInputField`, `ScrubInputDisplay` oder benutzerdefinierten Nachkommen, die Bearbeitungs- und Scrubbingzustand benĂ¶tigen.
