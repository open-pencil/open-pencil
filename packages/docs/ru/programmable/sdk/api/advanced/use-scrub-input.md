# useScrubInput

`useScrubInput()` читает локальный контекст scrub-ввода, предоставляемый `ScrubInputRoot`.

Используйте внутри `ScrubInputField`, `ScrubInputDisplay` или кастомных потомков, которым нужно состояние редактирования и скрабинга.
