# useNodeFontStatus

`useNodeFontStatus(node)` zwraca informacje o brakującej czcionce dla gettera węzła tekstowego.

Użyj go w panelach typografii i ostrzeżeniach, które muszą wskazywać niedostępne rodziny czcionek.
