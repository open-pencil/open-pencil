# useFillPicker

`useFillPicker(fill, onUpdate)` restituisce lo stato della categoria di riempimento derivato più gli helper di conversione per passare tra riempimenti solidi, a gradiente e immagine.

Usalo quando costruisci selettori di riempimento personalizzati oltre a `FillPickerRoot`.
