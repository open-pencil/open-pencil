# useFillPicker

`useFillPicker(fill, onUpdate)` retourne l'état de catégorie de remplissage dérivé ainsi que des helpers de conversion pour basculer entre les remplissages solides, dégradés et images.

Utilisez-le pour construire des sélecteurs de remplissage personnalisés au-delà de `FillPickerRoot`.
