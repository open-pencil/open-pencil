# useScrubInput

`useScrubInput()` odczytuje lokalny kontekst pola przeciągania dostarczany przez `ScrubInputRoot`.

Użyj go wewnątrz `ScrubInputField`, `ScrubInputDisplay` lub niestandardowych potomków, które potrzebują stanu edycji i przeciągania.

## Powiązane API

- [ScrubInputRoot](../components/scrub-input-root)
- [ScrubInputField](../components/scrub-input-field)
- [ScrubInputDisplay](../components/scrub-input-display)
