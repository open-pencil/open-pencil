# 国際化対応 (i18n) 仕様

## 概要

`@nanostores/i18n` を用いた多言語対応です。このリポジトリでは日本語ロケール (`ja`) を追加しています。

## 対応言語

`packages/vue/src/i18n/locale.ts:AVAILABLE_LOCALES`:

| コード | 表示名 | ロケールファイル |
|---|---|---|
| `ja` | 日本語 | `packages/vue/src/locales/ja.json` (このリポジトリで追加) |
| `en` | English | (デフォルト、コード内に埋め込み) |
| `de` | Deutsch | `de.json` |
| `es` | Español | `es.json` |
| `fr` | Français | `fr.json` |
| `it` | Italiano | `it.json` |
| `pl` | Polski | `pl.json` |
| `ru` | Русский | `ru.json` |
| `zh-CN` | 中文(简体) | `zh-CN.json` |

## デフォルト言語

`packages/vue/src/i18n/locale.ts`:

```typescript
export const localeSetting = atom<Locale | undefined>('ja')
```

**デフォルトは日本語 `ja`** に設定。
ユーザーは UI の言語メニューから切り替え可能。設定は `localStorage` (`inkly-locale`) に保存される。

## 翻訳ポリシー

UI の崩れ・可読性のため、すべてを日本語化せず、以下の使い分けをしています。

### 英語のまま残す項目

短く、アイコンと組み合わせて意味が明確な UI 要素は英語のままにする。

| カテゴリ | 例 |
|---|---|
| トップメニュー | `File`, `Edit`, `View`, `Object`, `Text`, `Arrange` |
| メニュー項目 (短い) | `Open`, `Save`, `Copy`, `Paste`, `Bold`, `Italic` |
| サイドバータブ | `Layers`, `Pages`, `Design`, `Code`, `AI`, `Assets` |
| プロパティパネル | `Position`, `Layout`, `Fill`, `Stroke`, `Effects`, `Export` |
| ツール名 | `Move`, `Frame`, `Rectangle`, `Ellipse`, `Pen`, `Text` |

理由: 日本語は欧文より幅が広く、短いコンテナで折り返しが発生する。アイコンと組み合わさるので意味の推測も容易。

### 日本語化する項目

長く、初見で意味が分かりにくい説明文・ダイアログメッセージは日本語化する。

| カテゴリ | 例 |
|---|---|
| 説明文 | 「変数コレクションがありません」「AI プロバイダーを接続してチャットを開始します」 |
| ダイアログのボタン | 「キャンセル」「適用」「閉じる」 |
| ヘルプテキスト | 「+ をクリックして混合した塗りを置換」 |
| エラーメッセージ | 「更新の確認に失敗: {error}」 |

## 翻訳ファイル構造

`packages/vue/src/locales/ja.json`:

```json
{
  "menu": {
    "file": "File",          // 英語のまま
    "open": "Open…",         // 英語のまま
    ...
  },
  "tools": {
    "move": "Move",          // 英語のまま
    ...
  },
  "panels": {
    "layers": "Layers",      // 英語のまま
    "noSelection": "選択されていません",  // 日本語化
    ...
  },
  "dialogs": {
    "cancel": "キャンセル",   // 日本語化
    "describeChange": "変更を入力…",  // 日本語化
    ...
  }
}
```

## 言語の切り替え方法

ユーザー操作: メニュー → `View` → `Language` → 日本語

プログラム:

```typescript
import { setLocale } from '@inkly/vue'
setLocale('en')  // 英語に切り替え
setLocale('ja')  // 日本語に切り替え
```

## 翻訳キーの追加方法

新しい UI 文字列を追加する場合:

1. 英語の翻訳キーをコード内に追加
   ```typescript
   // packages/vue/src/i18n/messages.ts
   export const myMessages = i18n('myCategory', {
     newKey: 'New Feature'
   })
   ```

2. `ja.json` に日本語訳を追加
   ```json
   {
     "myCategory": {
       "newKey": "新機能"
     }
   }
   ```

3. その他の言語ファイルにも追加 (任意)

4. パッケージを再ビルド
   ```bash
   bun --filter @inkly/vue build
   ```

## 追加した日本語翻訳の範囲

`ja.json` は 350 行で以下のカテゴリをカバー:

- `menu` (52 項目)
- `commands` (33 項目)
- `tools` (11 項目)
- `panels` (148 項目)
- `pages` (4 項目)
- `dialogs` (78 項目)
- `variableTypes` (8 項目)

## トラブルシューティング

### UI が英語のまま

- `localStorage` で `inkly-locale` の値を確認
- `setLocale('ja')` を Console から実行
- `packages/vue/src/locales/ja.json` がビルドに含まれているか確認 (`packages/vue/dist/locales/`)

### 文字が崩れる

- 日本語化したテキストが UI コンテナをはみ出していないか確認
- 該当箇所を英語に戻すか、テキストを短くする
