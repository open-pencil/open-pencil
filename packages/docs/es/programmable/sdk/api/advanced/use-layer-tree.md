# useLayerTree

`useLayerTree()` lee el contexto local del árbol de capas proporcionado por `LayerTreeRoot`.

Úsalo dentro de descendientes personalizados del árbol de capas que necesiten acceso a los elementos del árbol, el estado de selección, el estado de expansión o los manejadores de filas.
