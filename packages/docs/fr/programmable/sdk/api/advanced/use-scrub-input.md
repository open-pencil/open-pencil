# useScrubInput

`useScrubInput()` lit le contexte de saisie par glissement local fourni par `ScrubInputRoot`.

Utilisez-le à l'intérieur de `ScrubInputField`, `ScrubInputDisplay` ou de descendants personnalisés qui ont besoin de l'état d'édition et de glissement.
