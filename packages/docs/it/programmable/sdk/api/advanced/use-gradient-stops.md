# useGradientStops

`useGradientStops(fill, onUpdate)` gestisce lo stato dello stop attivo, il cambio di sottotipo, il trascinamento degli stop, e gli aggiornamenti di colore o opacità per i riempimenti a gradiente.

Usalo quando costruisci editor di gradienti personalizzati oltre alle primitive incluse nel pacchetto.
