# useNodeFontStatus

`useNodeFontStatus(node)` gibt fehlende-Schrift-Informationen für einen Textknoten-Getter zurück.

Verwenden Sie es in Typografie-Panels und Warnungen, die nicht verfügbare Schriftfamilien anzeigen müssen.
