# ToolbarItem

`ToolbarItem` expose l'état actif et le comportement de sélection pour un outil de la barre d'outils.

Utilisez-la à l'intérieur de `ToolbarRoot` quand vous voulez un balisage de bouton personnalisé mais un câblage de sélection d'outil partagé.
