# useFillPicker

`useFillPicker(fill, onUpdate)` devuelve el estado derivado de la categoría del relleno más helpers de conversión para cambiar entre rellenos sólidos, degradados e imágenes.

Úsalo cuando construyas selectores de relleno personalizados más allá de `FillPickerRoot`.
