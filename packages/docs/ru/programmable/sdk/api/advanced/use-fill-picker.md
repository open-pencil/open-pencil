# useFillPicker

`useFillPicker(fill, onUpdate)` возвращает производное состояние категории заливки и хелперы конверсии для переключения между сплошными, градиентными и графическими заливками.

Используйте его при создании кастомных пикеров заливки за пределами `FillPickerRoot`.
