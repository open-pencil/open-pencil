# useInlineRename

`useInlineRename(onCommit)` gestisce lo stato di modifica, il focus, la gestione dei clic esterni e il comportamento della tastiera per i flussi di rinomina inline.

Usalo in liste di pagine personalizzate, alberi dei layer o UI simili con rinomina sul posto.
