# LayerTreeRoot

`LayerTreeRoot` — SDK-примитив для рендеринга дерева слоёв с разметкой и стилями на стороне приложения.

Используйте его, когда нужна переиспользуемая структура дерева и подключение взаимодействий без встроенного представления.
