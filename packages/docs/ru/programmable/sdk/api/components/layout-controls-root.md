# LayoutControlsRoot

`LayoutControlsRoot` предоставляет контракт слота, возвращаемый `useLayout()`, в виде структурного примитива.

Используйте его, когда нужна переиспользуемая оболочка элементов управления лейаутом с разметкой на стороне приложения.
