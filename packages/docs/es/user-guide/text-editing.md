# Edición de texto

## Crear texto
Pulsa **T**, luego haz clic en el lienzo. Empieza a escribir inmediatamente.

## Edición inline
Doble clic en un nodo de texto para entrar en modo edición. Clic fuera para confirmar.

## Navegación del cursor
| Acción | Mac | Windows / Linux |
|--------|-----|-----------------|
| Izquierda/derecha | ← / → | ← / → |
| Arriba/abajo | ↑ / ↓ | ↑ / ↓ |
| Por palabra | ⌥ ← / ⌥ → | Ctrl + ← / Ctrl + → |
| Inicio/fin de línea | ⌘ ← / ⌘ → | Inicio / Fin |

**Shift** extiende la selección.

## Formato enriquecido
| Acción | Mac | Windows / Linux |
|--------|-----|-----------------|
| Negrita | ⌘ B | Ctrl + B |
| Cursiva | ⌘ I | Ctrl + I |
| Subrayado | ⌘ U | Ctrl + U |

## Selector de fuentes
Búsqueda, vista previa y scroll virtual. Fuentes del sistema en desktop (Tauri), Local Font Access API en el navegador.
